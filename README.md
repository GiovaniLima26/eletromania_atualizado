# Eletromania Distribuidora 🔌

Vitrine digital desenvolvida para a Eletromania Distribuidora, empresa de materiais elétricos localizada em Vera Cruz do Oeste - PR.

## 🚀 Acesse o site

[eletromaniadistribuidora.com.br](https://www.eletromaniadistribuidora.com.br)

## 🛠️ Tecnologias utilizadas

- [Next.js](https://nextjs.org/) + TypeScript
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/) (PostgreSQL + Storage)
- [Vercel](https://vercel.com/) (deploy)

## ✨ Funcionalidades

- Catálogo de produtos com filtro por categorias e busca
- Redirecionamento para WhatsApp ao clicar no produto
- Painel administrativo (`/admin`) para gerenciar produtos
- Upload de imagens via câmera ou galeria
- Design acessível e responsivo
